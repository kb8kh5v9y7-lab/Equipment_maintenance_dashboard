![Dashboard](Screenshot%202026-07-17%20193132.png)

## Project Overview

This project analyzes manufacturing equipment maintenance data using PostgreSQL and Power BI.

The dashboard provides insights into:

- Equipment downtime
- Repair performance
- Failure trends
- Technician workload
- Manufacturing area performance

## Business Questions

- Which manufacturing area has the highest downtime?
- Which failure codes create the most downtime?
- Which technicians perform the most repairs?
- How does downtime change over time?
- Which areas experience the most failures?
